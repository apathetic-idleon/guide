## ‚úÖ Developer Certificate of Origin (DCO)

To confirm the above in pull requests, please sign off your commits with:

```bash
git commit -s -m "Your commit message"
```

This adds a line to your commit message:

```
Signed-off-by: Your Name <you@example.com>
```

The sign-off certifies that you have the right to contribute and agree to the project‚Äôs licensing terms.
